## Model Parameters

- **Temperature** (0-2): Controls randomness
  - 0 = Deterministic, focused responses
  - 0.7 = Balanced creativity (default)
  - 2 = Very creative, diverse responses

- **Top P** (0-1): Nucleus sampling threshold
  - 0.9 = Recommended default
  - Higher values = more diverse word choices

- **Max Tokens**: Maximum response length (0 = unlimited)
